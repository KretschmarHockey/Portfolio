Taxi Booking System

For Web Development paper I made a taxi booking system. This was designed for admin and client use. The client site allows the user to put in details to order a taxi. That information is then stored in a database. The admin site can then view all taxi requests. They can then type in the booking's reference number to claim that booking.

This was achieved using PHP and javascript. Therefore, the website must be hosted on a server with PHP. Database login settings must be changed as well.
